# Fexend

Fexend is a collection of tools and products focused on user interface design and developer experience, including an admin template and a Visual Studio Code theme.

This GitHub organization hosts the source code, configurations, and releases related to these products.

## Projects

### Admin Template
A ready-to-use admin dashboard template that helps developers build internal tools and web applications faster. The template emphasizes clean design, responsive layout, and usability.

### VSCode Theme
A custom Visual Studio Code color theme designed for clarity, consistency, and visual comfort during long coding sessions.

## What You’ll Find Here

This organization contains:
- The admin template source code
- VSCode theme source and configuration
- Documentation and examples
- Project assets, styles, and build scripts

Some repositories are actively maintained, while others may be experimental or in-progress.

## Goals

- Provide a professional admin UI template that can be easily customized for different projects.
- Offer a visually pleasing and developer-friendly code editor theme.
- Share code and tooling openly with the community.

## Who It’s For

Fexend tools and themes are ideal for:
- Developers building dashboard interfaces
- Teams that want a consistent UI starting point
- Developers looking for a modern VSCode theme

## Notes

Repositories may vary in maturity and stability. Check individual project READMEs for usage and setup instructions.
